# v16.3 — SDR determinístico e seguro

Esta versão estabiliza o agente SDR após os testes do automático.

## Mudanças principais

- O SDR deixa de usar o histórico inteiro como fonte de decisão.
- A decisão passa a usar principalmente o estado salvo da conversa + a última mensagem recebida.
- O Gemini não reescreve as respostas por padrão. O fluxo usa mensagens fixas e previsíveis.
- A IA/Gemini pode ser reativada para lapidar texto com `NEXTLEAD_SDR_USE_GEMINI_POLISH=true`, mas não é recomendado agora.
- O fluxo fica mais linear:
  1. Tipo de negócio
  2. Presença atual: site/página ou Instagram/WhatsApp
  3. Objetivo: captar mais orçamentos/clientes pelo WhatsApp
  4. Momento: quer protótipo agora ou está pesquisando
  5. Handoff para vendedor

## Observação importante

Para testar direito, use um contato novo ou limpe o estado antigo em `sdr_states`, porque as conversas anteriores ficaram contaminadas pelos testes das versões anteriores.
